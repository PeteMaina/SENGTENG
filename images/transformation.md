# Project Transformation Strategy: Senteng Fashions Enterprise Upgrade

**From:** A static, monolithic HTML brochure website.

## 3. Detailed Transformation Breakdown

### A. Frontend Evolution (The "Face")
*   **Current:** `xmnlns.html` mixed CSS, content, and structure.
*   **Future:** Component-Based Architecture with **Material UI**.
    *   **"The Basic" Setup:** A clean React installation focusing on standard patterns rather than specialized build tools.
    *   **MUI Integration:** Deep integration of MUI components for a premium, professional "Enterprise" look.
    *   **Theming:** Create a custom MUI theme to match the Senteng branding (Green/White/Black).
    *   **State Management:** Use `React Context` or `Zustand` for global state (cart, auth).
    *   **Routing:** `React Router 6+` for handling SPA navigation.

### B. Backend Evolution (The "Brain")
*   **Current:** None (Static content).
*   **Future:** RESTful API.
    *   **Auth:** JWT (JSON Web Tokens) with secure HTTP-only cookies.
    *   **Endpoints:** 
        *   `/products` (CRUD)
        *   `/orders` (Transactions)
        *   `/users` (Profile management)
        *   `/quote-requests` (for bulk uniforms)

### C. Feature Expansion (The "Value")
1.  **Dynamic Product Catalog:** Instead of hardcoded images, fetch products from the DB.
2.  **Admin Dashboard:** A secure area using MUI DataGrid to upload new designs and view orders.
3.  **Interactive Quote Builder:** A multi-step MUI Stepper for companies to request bulk uniform orders.
4.  **Customer Reviews:** Real, database-stored reviews with MUI Rating components.
